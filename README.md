# DESAIN PROGRAM
Program dirancang dapat melakukan CRUD untuk barang toko dagangan. Program menggunakan java swing sebagai GUI. Program ini memiliki 2 class, yaitu.
## 1. Class Product
Class ini berisikan data dari stok barang yang terdapat pada toko. Atribut yang terdapat pada class ini awalnya adalah (sesuai soal):
- id
- nama
- harga
- kategori
  
Saya menambahkan atribut baru, yaitu.
- stok

## 2. Class ProductMenu
class ini berisikan GUI dari program, yang bertujuan untuk mengatur GUI dan melakukan method seperti insert, delete, update, dan filter.

# PENJELASAN ALUR
## 1. Tampilan utama
Menampilkan tabel dari data barang dan menampilkan menu untuk aksi lainnya.

## 2.Penambahan Data Baru
User dapat mengisi data dengan mengisi form secara langsung.

## 3. Update data
User dapat mengubah data dengan memilih salah satu data yang terdapat pada tabel dan mengganti langsung pada form.

## 4. Delete data
User dapat menghapus dengan memilih data pada tabel dan menekan tombol delete.

## 5. Filter
User dapat menampilkan data sesuai kategori yang dipilih.
